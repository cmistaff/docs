# CrestPet System

CrestPet System is a comprehensive, multi-tenant veterinary practice management platform. It serves veterinary clinics, pet owners, enterprise organizations, and platform administrators through seven portals (five core portals plus the Marcia AI assistant and Vendor purchase order confirmation portal), each tailored to a specific user group.

## What CrestPet Does

CrestPet provides a unified platform for managing every aspect of veterinary care:

- **Appointment scheduling and management** with real-time availability, slot reservations, and video consultations
- **Electronic health records** including consultation notes, dental charts, diagnostic requests, imaging, treatment sheets, and anesthetic sheets
- **Pet wellness and health scoring** with AI-powered health assessments, wellness plans, and wellness reports with video summaries
- **Inventory and purchasing** with stock tracking, lot management, low-stock alerts, auto-reorder, and vendor/purchase order workflows
- **Billing and payments** with Stripe Connect integration, invoicing, insurance splits, cash payments, payout tracking, and staff commissions
- **Multi-location enterprise management** with analytics dashboards, predictive analytics, performance monitoring, and centralized clinic oversight
- **Consumer pet owner portal** with appointment booking, health records access, AI-powered symptom checking, medication tracking, and vaccination history
- **Referral network** for inter-clinic patient referrals with availability-based scheduling
- **WhatsApp integration** for client communication via Marcia, the AI veterinary assistant
- **Document management** with DocuSign templates, consent forms, legal policies, and PDF certificate generation

CrestPet is organized into seven portals, each accessed through a subdomain:

## Tech Stack

| Layer | Technology |
|-------|-----------|
| Frontend | Next.js 15 (App Router), React 19, TypeScript 5.8, Tailwind CSS 4 |
| Backend | NestJS 11, TypeScript 5.8, Prisma 6 |
| Database | PostgreSQL (RDS) |
| Cache | Redis (ElastiCache) |
| Auth | Better Auth + JWT + Prisma adapter, Passkey/WebAuthn support |
| Runtime | Node.js 22, pnpm 10 |
| Infrastructure | AWS ECS Fargate, EKS (multi-region), RDS, ElastiCache, S3, CloudFront, ALB, WAF |
| API Client | Orval (auto-generated from OpenAPI spec) |
| Testing | Bun test runner (server unit + E2E), Playwright (frontend E2E) |
