# Marcia Mobile App

Marcia is the consumer-facing mobile application for CrestPet, built with Expo React Native. It gives pet owners a direct line to their veterinary clinic and an AI-powered pet health assistant.

## What Marcia Does

<CardGroup cols={2}>
  <Card title="Appointment Booking" icon="calendar">
    Search clinics, view real-time availability, and book appointments directly from the app.
  </Card>
  <Card title="Health Records" icon="file-medical">
    Access your pet's complete health history, vaccination records, and treatment notes.
  </Card>
  <Card title="AI Health Assistant" icon="robot">
    Ask Marcia about pet symptoms, medication questions, or general pet care. Powered by AI with veterinary knowledge.
  </Card>
  <Card title="Medication Tracking" icon="pills">
    Track medication schedules, get reminders, and view prescription history.
  </Card>
</CardGroup>

## Tech Stack

| Layer | Technology |
|-------|-----------|
| Framework | Expo (React Native) |
| Language | TypeScript |
| Backend | Express.js |
| AI | Custom veterinary LLM (Marcia Code) |
| Push Notifications | OneSignal |
| Messaging | WhatsApp via Twilio |

## Repository

The mobile app lives in `cmistaff/AppCrestPetConsumer`. The Express.js backend handles API requests, AI inference, and push notification delivery.
