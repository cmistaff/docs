## Rate card

| Model | Model ID | Best for | Input ($ / 1M tokens) | Output ($ / 1M tokens) |
|---|---|---|---|---|
| **Marcia Vet** | `marcia-vet-v1` | Flagship vet reasoning, multi-step clinical workflows, treatment planning | **$2.50** | **$7.50** |
| **Marcia Vet Mini** | `marcia-vet-mini` | High-volume vet chat, autocomplete, in-app helpers, consumer pet-owner Q&A | **$0.50** | **$2.00** |

Both models share the same authentication, endpoint, and request shape — pick the model by passing it in the `model` field of the request.

## Free tier

Every Ask Marcia account includes a **free monthly allowance of 500,000 tokens combined across both models**. No credit card required to start. The allowance resets at the start of each calendar month (UTC).

Once you exceed the free allowance, usage is billed at the per-token rates above. Add a payment method to unlock paid usage; otherwise additional requests return `402 Payment Required` until the next reset.

## How billing works

- **Token counting:** Input and output tokens are counted separately and at the rate of the model the request was served on.
- **Model is final:** The `marcia-vet-*` models do not silently downgrade or upgrade. If you request `marcia-vet-v1` you are billed at the `marcia-vet-v1` rate even if a fallback provider serves the response.
- **Failed requests are not billed:** Requests that error before generating any output are not charged.
- **Streaming requests:** Billed identically to non-streaming on the same model — token count is final at stream completion.

## Worked example

A vet clinic using Marcia Vet Mini for in-app pet-owner Q&A and Marcia Vet for treatment-planning reviews:

| Workload | Model | Input tokens | Output tokens | Cost |
|---|---|---|---|---|
| Pet-owner Q&A | `marcia-vet-mini` | 6,000,000 | 1,500,000 | (6 × $0.50) + (1.5 × $2.00) = **$6.00** |
| Treatment-plan reviews | `marcia-vet-v1` | 800,000 | 250,000 | (0.8 × $2.50) + (0.25 × $7.50) = **$3.88** |
| **Total** | | **6.8M in, 1.75M out** | | **$9.88** |

The first 500,000 tokens combined are free, so the actual invoice would be **$9.88 minus the free-tier value** (computed at the blended rate of the models consumed inside the free allowance).
