> **Source Branch:** `prestaging`

Standards for charts, graphs, and data display in the Crest Pet System.

## 📚 Chart Library

**Primary:** Recharts (React wrapper for D3)

```
import {
  LineChart, BarChart, PieChart, AreaChart,
  Line, Bar, Pie, Area,
  XAxis, YAxis, CartesianGrid, Tooltip, Legend
} from "recharts"
```

## 📊 Chart Type Selection

| Data Type | Chart Type | Use Case |
| --- | --- | --- |
| Comparison | Bar Chart | Compare values across categories |
| Composition | Pie/Donut | Show parts of a whole |
| Trend | Line Chart | Show change over time |
| Volume | Area Chart | Show cumulative values |
| Distribution | Histogram | Show frequency distribution |

## 🥧 Pie Chart Guidelines

### Best Practices

- Maximum 6-8 segments
- Minimum 5% per segment (combine smaller into "Other")
- Always include percentages
- Use consistent ordering (largest to smallest)

### Donut vs Pie

| Type | Usage |
| --- | --- |
| Pie | Simple composition |
| Donut | When showing total in center |

---

## 🔢 Number Formatting Utilities

```
// Currency with abbreviation
formatCurrency(1500000) // "$1.5M"

// Plain number with commas
formatNumber(1234567) // "1,234,567"

// Compact notation
formatCompact(1500) // "1.5K"
```
