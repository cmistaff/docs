## 🏗️ System Architecture Overview

### Multi-Tenant Portal System

The Crest Pet System is a **multi-tenant veterinary platform** with 6 distinct portals:

| Portal | URL Pattern | Users |
| --- | --- | --- |
| Clinic | clinic.{tenant}.crestpet.com | Vet Staff |
| Consumer | consumer.{tenant}.crestpet.com | Pet Owners |
| Enterprise | enterprise.{tenant}.crestpet.com | Corporate |
| Platform | admin.crestpet.com | Crest Admin |
| Referral | referral.{tenant}.crestpet.com | Partners |
| Marcia AI | marcia.crestpet.com | AI Chat |
