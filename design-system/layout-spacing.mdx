# Layout & Spacing

> **Source Branch:** `prestaging`
> **Last Updated:** December 2024

Grid systems, spacing standards, and responsive patterns for the Crest Pet System.

---

## 🏗️ Standard Dashboard Layout

### Structure

| Element | Size | Notes |
| --- | --- | --- |
| Header | `h-16` (64px) | Fixed at top |
| Sidebar (expanded) | `w-72` (288px) | Collapsible |
| Sidebar (collapsed) | `w-20` (80px) | Icon-only |
| Main content | `flex-1` | Scrollable |
| Max content width | `max-w-[1600px]` | Centered |

### Responsive Behavior

| Breakpoint | Sidebar | Header |
| --- | --- | --- |
| Mobile (<768px) | Hidden (drawer) | Hamburger menu |
| Tablet (768-1024px) | Collapsed | Full |
| Desktop (>1024px) | Expanded | Full |

## 📏 Spacing Scale

### Standard Gaps

| Token | Pixels | Usage |
| --- | --- | --- |
| `gap-1` | 4px | Tight grouping |
| `gap-2` | 8px | Related items |
| `gap-3` | 12px | Form fields |
| `gap-4` | 16px | Card content |
| `gap-6` | 24px | Sections |
| `gap-8` | 32px | Major sections |

### Responsive Spacing

```
className="gap-4 md:gap-6 lg:gap-8"
```
