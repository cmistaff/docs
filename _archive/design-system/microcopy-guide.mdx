# Microcopy Guide

> **Source Branch:** `prestaging`
> **Last Updated:** December 2024

Voice, tone, and writing standards for the Crest Pet System.

---

## 🎤 Brand Voice

### Core Attributes

| Attribute | Description |
| --- | --- |
| Warm | Caring and supportive, especially about pets |
| Professional | Expert and trustworthy |
| Clear | Simple, jargon-free language |
| Efficient | Concise, respects user's time |

---

## 🎯 Portal-Specific Tones

### Consumer Portal

- **Tone:** Friendly, reassuring, supportive
- **Reading Level:** 8th grade
- **Example:** "Your pet's health is our priority. Let's get started!"

### Clinic Portal

- **Tone:** Professional, efficient, precise
- **Reading Level:** Professional
- **Example:** "Patient record updated. View history."

### Enterprise Portal

- **Tone:** Strategic, data-driven, premium
- **Reading Level:** Business professional
- **Example:** "Network performance metrics updated for Q4."

## 🔘 Button Labels

### Principles

- Use action verbs (Add, Save, Create, Send)
- Be specific (not "Submit" but "Save Changes")
- Keep to 1-3 words

### Common Actions

| Action | Good | Avoid |
| --- | --- | --- |
| Create | "Add Patient" | "Create New" |
| Save | "Save Changes" | "Submit" |
| Delete | "Remove" | "Delete Forever" |
| Cancel | "Cancel" | "Go Back" / "Nevermind" |
| Confirm | "Confirm" | "Yes, I'm sure" |

## ❌ Error Messages

### Formula

**What happened** + **Why** + **What to do next**

### Examples

| Bad | Good |
| --- | --- |
| "Error" | "Unable to save. Check your connection and try again." |
| "Invalid input" | "Email address is invalid. Please enter a valid email." |
| "Failed" | "Payment failed. Your card was declined. Try a different card." |

### Validation Errors

| Field | Message |
| --- | --- |
| Required | "[Field] is required" |
| Email | "Please enter a valid email address" |
| Phone | "Please enter a valid phone number" |
| Password | "Password must be at least 8 characters" |
| Match | "Passwords don't match" |

## 📭 Empty State Copy

### No Data Yet

> **Title:** No [items] yet
> **Description:** [Action] to get started.
> **CTA:** Add [Item]

Example:

> **No patients yet**
> Add your first patient to get started managing their health records.

### No Search Results

> **Title:** No results found
> **Description:** Try adjusting your search or filters.
> **CTA:** Clear Filters

### Premium Feature

> **Title:** [Feature] is a premium feature
> **Description:** Upgrade your plan to access [benefit].
> **CTA:** View Plans

## ✅ Success Messages

### Principles

- Confirm what action was completed
- Include relevant context (name, ID)
- Keep brief
- Use past tense

### Examples

| Action | Message |
| --- | --- |
| Create | "Patient 'Max' added successfully" |
| Update | "Changes saved" |
| Delete | "Patient removed" |
| Send | "Invitation sent to john@email.com" |

---

## 🏷️ Form Labels

### Guidelines

- Use sentence case ("Email address" not "Email Address")
- Be specific ("Work phone" not just "Phone")
- Mark required fields with asterisk (*)
- Mark optional fields with "(optional)"

### Placeholders

| Do | Don't |
| --- | --- |
| "Enter email address" | "email@example.com" |
| "Search patients..." | "Type here" |
| "Select an option" | "Choose" |

## 📅 Date & Time Formatting

| Format | Usage |
| --- | --- |
| Dec 8, 2024 | Display dates |
| 2:30 PM | Display times |

| Today, 2:30 PM | Relative dates |
| 3 hours ago | Recent timestamps |

## 🔢 Number Formatting

| Type | Format | Example |
| --- | --- | --- |
| Currency | $X,XXX.XX | $1,234.56 |
| Large numbers | X.XK, X.XM | 1.2K, 3.5M |
| Percentages | XX% | 85% |
| Phone | (XXX) XXX-XXXX | (555) 123-4567 |
